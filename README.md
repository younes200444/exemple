# Brasserie Jo De Bruges

Site démo premium pour la brasserie Jo De Bruges — 23 Avenue de la Gineste, 12000 Rodez.

## Démarrage

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Stack

Next.js 15 · TypeScript · Tailwind CSS · Framer Motion · Lucide Icons
